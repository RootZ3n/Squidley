# Skill: weather-reporting
## Purpose
Provides guidelines and best practices for reporting current and forecasted weather conditions.
## Commands and Responses
- **Command:** "What's the weather like today?"
  - **Response:** "The current temperature is [temperature] degrees, with [weather condition]."
- **Command:** "Will it rain tomorrow?"
  - **Response:** "Tomorrow’s forecast predicts a chance of [rain percentage]% for precipitation."
